Flipr

このモジュールについて

Fliprモジュールは、プレゼンテーションを最重要視して開発した非常にシンプルなフォトアルバムモジュールです。 視覚的なアルバム一覧や、モーダルウィンドウによる画像の表示、スライドショーなどの機能を提供します。

機能説明

  • 画像アップロード機能(gif, jpeg, pngに対応)
  • サムネイル機能(GDを使用)
  • アルバム管理機能
  • スライドショーブロック
  • モーダルウィンドウによる画像のポップアップ
  • NiceAdminモジュールのAdminTaskBarに対応
  • シンプルで直感的な操作性

依存関係

  • PHP5.2.0以上
  • PHPのGD拡張が利用可能であること
  • MySQL5以上(InnoDBが使用可能な環境が推奨されます)
  • このモジュールはPenginフレームワークが必要です。
  • AdminTaskBar機能を利用するには、NiceAdminモジュールのインストールが必要です。

インストール方法

  1. アーカイブを展開して、flipr を html/modules/ の下にコピー
  2. モジュール管理でインストールを実行してください。
  3. アクセス権限をこのモジュールの機能を使用するユーザに与えてください。

トラブルシューティング

アルバム作成時や写真アップロード時に「画像アップロード先のディレクトリが存在しません。」というエラーが出ます

原因

画像保存用（アップロード用）のフォルダが無いか、書き込みできない状態になっています。

Fliprモジュールはアップロードフォルダがない場合、自動でフォルダを作成しようとします。 しかし、何らかの環境的な要因(アクセス権限、PHPの設定など)でアップロードフォルダの作成に失敗することがあります。 その場合は、FTPやSSHなどでサーバにアクセスし、手動でフォルダを作成する必要があります。 なお、エラーの詳細は「全般設定」でデバッグモードを「PHPデバッグ」にすることで、確認することができます。

対策

html/uploadsにfliprという名前のフォルダが無ければ、フォルダを作成してください。

html/uploads/fliprのパーミッションが777以外の場合は、パーミッションを777(書き込み可能)に設定してください。
